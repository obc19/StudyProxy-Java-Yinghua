# StudyProxy-Java

英华学堂自动学习工具 - Java版

## 功能特性

- ✅ 多账号并行刷课
- ✅ 多课程并行学习（每账号5课程）
- ✅ 自动验证码识别（ddddocr）
- ✅ 视频自动学习
- ✅ 考试/作业自动答题（AI支持）
- ✅ 账号密码错误自动退出
- ✅ 配置文件驱动

## 快速开始

### 环境要求

- JDK 1.8+
- Maven 3.6+
- Python 3.8+ (OCR服务)
